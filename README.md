# AI/ML Task 2 – Feature Engineering & Model Optimization

## Project Overview
This project builds machine learning models to predict house prices using the California Housing Dataset.

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Steps Performed
- Data preprocessing
- Feature scaling
- Train-test split
- Model training
- Performance evaluation using RMSE and R²
- Model comparison

## Result
Decision Tree model achieved the best performance with highest R² score and lowest RMSE.

## Tools Used
Python, pandas, NumPy, scikit-learn, matplotlib, Jupyter Notebook
